# Developer Guide

## Local Development

### Setup
```bash
make install
```

### Run tests
```bash
make test
```

### Lint
```bash
make lint
```

### Run ingestion loop locally
```bash
make run
```

## Deployment

### Deploy to Raspberry Pi
```bash
make deploy
```

### Restart service
```bash
make restart
```

### Tail logs
```bash
make logs
```

### Inspect database
```bash
make db-shell
```

## Makefile Help

```bash
make help
```

Displays all available commands with descriptions.
